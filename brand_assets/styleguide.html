Scroll Reveal

Elements below the fold are observed via IntersectionObserver. When they enter the viewport, will-animate is added, then visible via double-RAF — ensuring elements are never invisible on initial render (important for screenshot compatibility).

.reveal.will-animate {
  opacity: 0;
  transform: translateY(28px);
  transition: opacity 500ms cubic-bezier(0.22, 0.61, 0.36, 1),
              transform 500ms cubic-bezier(0.22, 0.61, 0.36, 1);
}
.reveal.will-animate.visible { opacity: 1; transform: translateY(0); }
.will-animate.stagger-1 { transition-delay:  80ms; }
.will-animate.stagger-2 { transition-delay: 160ms; }
